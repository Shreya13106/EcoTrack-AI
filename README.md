# 🌍 EcoTrack AI

## AI-Powered Carbon Footprint Tracker & Sustainability Assistant

EcoTrack AI is a web-based platform that helps users calculate their carbon footprint, visualize emission sources, and receive AI-powered sustainability recommendations.

---

## 🚀 Features

- 📊 Carbon Footprint Calculator
- 🌱 Eco Score Analysis
- 📈 Emission Breakdown Visualization using Chart.js
- 🤖 AI Sustainability Assistant powered by Groq
- 📱 Responsive Dashboard UI

---

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript
- Chart.js

### Backend
- Python
- Flask
- Flask-CORS

### AI Integration
- Groq API
- Llama 3.3 70B Versatile

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/EcoTrack-AI.git
cd EcoTrack-AI
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Create .env File

```env
GROQ_API_KEY=YOUR_GROQ_API_KEY
```

### 4. Run Backend

```bash
python app.py
```

### 5. Open Frontend

Open:

```text
frontend/index.html
```

in your browser.
